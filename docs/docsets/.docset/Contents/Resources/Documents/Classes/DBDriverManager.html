DBDriverManager

open class DBDriverManager

The driver manager. The driver manager maintains a list of currently registered drivers. When a connection to a database is required then a call to the driver manager’s connect(url:username:password:properties:) method will locate the appropriate driver for the given URL and then create and return a connection object.

Drivers are registered by calling their DBDriver.register() class method.
